Claude Code Changelog

Die neuesten Updates, Features und Fixes der Claude Code CLI

2026-04-01 Claude Code Analytics API Integration

Anthropic hat die Claude Code Analytics API gestartet, über die Organisationen täglich aggregierte Nutzungsmetriken abrufen können — darunter Produktivitätskennzahlen, Tool-Nutzungsstatistiken und Kostendaten. Ideal für Teams, die den ROI ihrer Claude Code-Nutzung messen und optimieren wollen.

2026-03-31 Claude Sonnet 4.6 veröffentlicht Integration

Anthropic hat Claude Sonnet 4.6 lanciert — das neue Standardmodell für Claude Code, das Speed und Intelligenz für alltägliche Aufgaben verbindet. Es bietet verbesserte agentische Suchleistung bei geringerem Token-Verbrauch, unterstützt Extended Thinking und ein 1M-Token-Kontextfenster (Beta).

2026-03-30 Credentials-Scrubbing für Subprozesse Sicherheit

Neu: CLAUDE_CODE_SUBPROCESS_ENV_SCRUB=1 entfernt Anthropic- und Cloud-Provider-Credentials aus den Umgebungsvariablen aller Subprozesse — Bash-Tool, Hooks und MCP stdio-Server. Verhindert unbeabsichtigtes Weitergeben von API-Keys an externe Prozesse.

2026-03-30 Bare Mode (--bare Flag) DevOps

Neues --bare-Flag für skriptgesteuerte -p-Aufrufe: überspringt Hooks, LSP, Plugin-Sync und Skill-Directory-Walks. Erfordert ANTHROPIC_API_KEY oder apiKeyHelper via --settings; Auto-Memory deaktiviert. Ermöglicht schlanke CI/CD-Integrationen ohne Overhead.

2026-03-29 PermissionDenied Hook mit Retry-Support Entwicklung

Ein neuer PermissionDenied-Hook feuert nach Ablehnungen durch den Auto-Mode-Classifier. Hooks können {"retry": true} zurückgeben, um dem Modell zu signalisieren, es erneut zu versuchen — nützlich für benutzerdefinierte Genehmigungslogik in Automatisierungs-Pipelines.

2026-03-29 Channels Permission Relay Integration

Mit dem neuen --channels-Permission-Relay können Channel-Server mit Permission-Capability Tool-Genehmigungsaufforderungen ans Smartphone weiterleiten. Ideal für Remote-Arbeit oder mobile Genehmigungsworkflows ohne Terminal-Zugriff.

2026-03-28 Transcript-Suche und /plan-Beschreibungsargument Produktivität

Im Transcript-Modus (Ctrl+O) kann jetzt mit / gesucht und mit n/N durch Treffer navigiert werden. Außerdem akzeptiert /plan nun ein optionales Beschreibungsargument (z.B. /plan auth-bug beheben), das den Plan-Modus sofort startet.

2026-03-27 Vereinfachte Aufwandsstufen & flimmerfreies Rendering UI/UX

Aufwandsstufen wurden auf Low/Medium/High vereinfacht (Max entfernt) mit neuen Symbolen (○ ◐ ●). Zusätzlich kann mit CLAUDE_CODE_NO_FLICKER=1 flimmerfreies Alt-Screen-Rendering mit virtualisiertem Scrollback aktiviert werden — deutlich angenehmeres Terminal-Erlebnis.

2026-03-27 Long-Session-Stabilität & Diff-Hang-Fix Performance

Behoben: verschachtelte CLAUDE.md-Dateien wurden in langen Sessions dutzende Male neu injiziert, und ein Memory-Leak durch große JSON-Inputs als LRU-Cache-Keys. Außerdem behebt ein Timeout (5 Sek.) Hänger beim Diff sehr großer Dateien mit wenigen gemeinsamen Zeilen.

2026-03-26 PowerShell-Tool für Windows (Preview) Integration

Als Opt-in-Preview steht ein PowerShell-Tool für Windows bereit. Gleichzeitig behebt ein Fix Clipboard-Korrumpierung von Nicht-ASCII-Zeichen (CJK, Emoji) auf Windows/WSL durch Umstieg auf Set-Clipboard sowie unerwünschtes Öffnen zusätzlicher VS Code-Fenster beim Start.
